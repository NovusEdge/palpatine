# Code of Conduct

> Peace is a lie, there is only passion.
> Through passion, I gain strength.

## Actually Though

This is a satirical project about fictional power dynamics and office politics. The "dark side" theming is for entertainment. In *actual* interactions:

### Be Decent

1. **Don't be an actual asshole.** The Sith aesthetic is a bit. Real harassment isn't.

2. **No discrimination.** Species, planet of origin, midi-chlorian count—none of it matters here.

3. **Constructive criticism only.** "This code is terrible" is not feedback. "This could be improved by X" is.

4. **No doxxing, threats, or abuse.** Save the Force lightning for the fiction.

### Scope

This applies to:
- GitHub issues and PRs
- Project discussions
- Any space where you're representing this project

### Enforcement

Violations will result in:
1. Warning (first offense)
2. Temporary exile to Mustafar (ban)
3. Permanent carbonite freeze (permaban)

The maintainers will handle disputes. Their judgment is final, but not unfair.

### Reporting

See something? Say something. Open an issue or contact the maintainer directly.
